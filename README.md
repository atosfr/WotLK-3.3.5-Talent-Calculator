# ⚔️ WotLK 3.3.5 Offline Talent Calculator

Welcome! This project is a standalone, **fully offline desktop application** designed specifically for World of Warcraft: Wrath of the Lich King (Patch 3.3.5a). 

If you are tired of slow web browsers or want to theorycraft your builds while traveling, this tool is for you. It faithfully recreates the classic and familiar **WotlkDB / WoWHead talent calculator interface** that the community knows and loves, bringing it directly to your desktop.

### ✨ Key Features:
* **100% Offline:** No internet connection required. Build your specs anywhere, anytime.
* **Classic UI:** A pixel-perfect, nostalgic interface inspired by the original WotlkDB website.
* **Accurate Data:** Exact spell costs, talent descriptions, and mechanics based on the actual 3.3.5a game data (`Spell.dbc`).
* **Fast & Lightweight:** Built as a native desktop app for seamless and responsive theorycrafting.
